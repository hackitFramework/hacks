# GitGrab
Search git repos for key text. Searches all commits, and code stored in the repo.
<br>

## Requirements

Python Version: python3.x <br>

```
bash install.sh
```
## Running
Navigate to:
```
cd /<path_to>/hacks/gitgrab
```
Run:

```
python gitgrab.py
```
```gitgrab.py``` has a ui that will handle the rest.
